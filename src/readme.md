## ⚙️ Setup

- In the project directory, ou can follow the steps below to download the dependencies:
  - Install with npm:
  ```sh
  npm i
  ```
  - Install with yarn:
  ```sh
  yarn install
  ```
- In the project directory, you can run:
  - Run with npm:
  ```sh
  npm start
  ```
  - Run with yarn:
  ```sh
  yarn start
  ```

## ⌨️ Usage

You need to enter a title and content to create a note.
